# 海水养殖水质分析系统 - 使用说明

## 系统简介
这是一个用于海水养殖水质监测数据分析的C语言程序，包含数据查询、预处理、修改、备份和统计分析功能。

## 快速开始

### 方法一：使用批处理脚本（推荐）
直接双击运行 `run.bat` 文件即可启动程序。

### 方法二：使用命令行
```bash
# 编译程序（如果没有编译过）
gcc -std=c11 -O2 -Wall -o water_analysis.exe main.c common.c dataset.c query.c modify.c backup.c statistics.c -lm

# 运行程序
water_analysis.exe
```

## 程序功能

### 主菜单
1. **数据查询与浏览** - 分页浏览、条件筛选、排序
2. **数据预处理** - 异常值检测、缺失值填充、移动平均滤波
3. **数据修改与删除** - 修改记录、删除记录
4. **数据备份与恢复** - 创建备份、恢复数据
5. **统计分析** - 基本统计、预警分析、相关性分析
0. **退出系统**

## 数据说明
- 数据文件位置：`data/data.csv`
- 数据包含：水温、盐度、pH、溶解氧、降水量、气温等参数
- 数据量：23,203 条监测记录

## 常见问题

### Q: 为什么输出是乱码？
A: 请确保控制台使用 UTF-8 编码。使用 `run.bat` 启动会自动设置正确的编码。

### Q: 如何重新编译程序？
A: 运行 `build.bat` 文件即可重新编译。

### Q: 程序生成了哪些文件？
A: 程序会生成以下文件：
- `data_overview.txt` - 数据概览
- `alerts.txt` - 预警信息
- `stat_report.txt` - 统计报告
- `filtered_w*.csv` - 滤波后的数据
- `backups/*.csv` - 备份文件
